# CS_373_Lab7_CI
This project demonstrates the implementation of a Continuous Integration (CI) pipeline using Jenkins on an AWS EC2 instance to automatically build a Java application. The project source code is hosted on GitHub and connected to Jenkins באמצעות secure SSH authentication. Maven is used as the build automation tool to compile the code, manage dependencies, and package the application using the "clean install" lifecycle. A GitHub webhook is configured to trigger Jenkins builds automatically whenever changes are pushed to the repository, enabling real-time integration and testing. This setup showcases key cloud computing and DevOps concepts, including automated builds, cloud-based infrastructure, version control integration, and event-driven workflows.
